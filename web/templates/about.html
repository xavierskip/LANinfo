局域网信息抓取工具

基于python抓取网络信息，信息存储在splite数据库中，通过Flask提供web服务

Usage

ini.py
相关配置文件，数据库路径，以及抓取的网段名称，以及在数据库中存储信息的表名（用来查询）

DATAPATH # 数据库路径

SCHEMA # 数据表结构

INFO # 初始导入数据

NETNAME # 连接的数据库表名

DEST # 抓取的目标网络 

SHOW # 1:显示用过的IP 0:显示没有用过的IP

scan.py

抓取指定网段的机器信息，(通过抓取的MAC地址来判断，那些IP地址被使用过)，并存储进数据库，存储的数据表名根据抓取的网段生成

do.bat

用来windows下定时抓取的脚本，抓取记录写入/data/log中

db.py

连接数据库，导入初始数据

backupDB.bat

Windows下定时备份数据库

app.py

提供web访问服务
